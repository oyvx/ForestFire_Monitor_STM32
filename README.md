# ForestFire_Monitor_STM32｜森林防火监测终端
## 项目概述
基于STM32F103 + BC28 NB-IoT模组搭建森林防火采集终端，使用DHT11、MQ2烟雾传感器、MQ7一氧化碳传感器采集林区环境数据，通过**MQTT协议**上传阿里云物联网平台；环境参数超标时驱动板载蜂鸣器触发火情报警，搭配Qt上位机订阅云端数据实现实时数值显示与动态曲线绘制，整套系统实现野外森林火情预警。

## 硬件组成
- 主控：STM32F103C8T6
- 通信：BC28 NB-IoT（串口AT指令入网，MQTT上传数据）
- 传感器：DHT11温湿度、MQ2烟雾浓度、MQ7一氧化碳浓度
- 外设：有源蜂鸣器（超限火警提醒）

## 通信链路
STM32采集 → BC28-NB-IoT → MQTT发布 → 阿里云IoT平台 → Qt/Android客户端MQTT订阅 → 数据展示&曲线绘图&告警
